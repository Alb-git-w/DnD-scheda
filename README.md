# Scheda D&D 5e

App mobile per gestire la scheda del personaggio in D&D 5a edizione. Funziona direttamente nel browser, installabile come PWA.

## Funzionalità

- 6 sezioni navigabili via swipe: Stato, Combattimento, Abilità, Magia, Inventario, Personaggio
- Setup guidato in 7 step per creare il personaggio
- Calcoli automatici: modificatori, bonus competenza, spell DC, iniziativa, tiri salvezza
- Gestione PF, dadi vita, slot incantesimo, condizioni, ispirazione
- Dati salvati in locale (localStorage) — niente account, niente server
- Installabile su iOS e Android come app standalone (senza barra del browser)

## Uso

Apri [la pagina GitHub Pages](https://alb-git-w.github.io/DnD-scheda/) sul telefono, poi:
- **iOS**: Safari → Condividi → Aggiungi a schermo Home
- **Android**: Chrome → menu ⋮ → Installa app

## Stack

HTML + CSS + JS vanilla. Nessuna dipendenza, nessun build step.
